## What is WeatherIO?

WeatherIO is a weather app frontend. It provides a search field for city lookup, a back button, and a location detection icon. The current weather is displayed with city name, region, and country. Material Icons are used for all UI icons. The default location shown is London.

---

## Features

- **City search** — type a city name and hit the search icon
- **Location detection icon** — UI element for current location lookup
- **Current weather display** — shows city, region, country
- **Material Icons** — Google Material Icons for clean, consistent iconography
- **Minimal, clean design** — focused layout with no distractions
